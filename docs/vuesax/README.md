## Install in project via NPM or YARN

```bash

# install
npm install vuesax # OR yarn add vuesax

```

</box>

<box>

## Use

Vuesax is a Vuejs library. To use it, add the code below:

```js
import Vue from 'vue'
import Vuesax from 'vuesax'

import 'vuesax/dist/vuesax.css' //Vuesax styles
Vue.use(Vuesax)
```
